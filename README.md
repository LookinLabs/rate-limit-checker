# Install

`git clone https://github.com/Sh1Yo/rate-limit-checker && cd rate-limit-checker && go build rate.go`

or

`go get github.com/Sh1Yo/rate-limit-checker`

# Usage

```

Usage of rate:
  -X string
        method (default "HEAD")
  -a int
        times (default 500)
  -s    continue after the code changing
  -t int
        threads (default 10)
  -u string
        link

```
